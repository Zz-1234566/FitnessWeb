# 登录页重新设计

## 风格
清新自然，居中卡片式，白色+蓝绿配色

## 布局
- 垂直居中白色圆角卡片（border-radius: 16px，box-shadow）
- 背景浅灰蓝渐变（#f0f5ff → #e6f7ff），不使用外部图片
- 卡片内边距 40px/32px

## Logo + 标题
- Logo 居中 64x64
- 标题"智能健身平台"深灰 #333
- 副标题"竭诚为您服务"浅灰 #999

## 表单
- 去掉 Tabs（只有一个登录方式）
- 输入框圆角 8px，高度 44px，聚焦边框蓝绿 #13c2c2
- 登录按钮蓝绿渐变（#36cfc9 → #13c2c2），圆角 8px，高度 44px
- 自动登录保留，注册链接居中放底部
- 去掉 ActionIcons 社交图标

## 动效
- 卡片入场：上浮淡入 translateY(20px) → 0，0.6s
- 输入框聚焦：边框过渡 + scale(1.01)
- 按钮悬停：颜色加深 + 阴影 + 上移 2px
- 按钮点击：scale(0.98)

## Footer
- 极简化或去掉
